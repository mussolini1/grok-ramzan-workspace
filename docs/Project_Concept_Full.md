# FNAF: Shadows Reckoning — Полная Концепция Проекта

## 1. Общая Идея
Фанатский хоррор-проект в вселенной Five Nights at Freddy’s, созданный на Python с использованием Ursina Engine. Цель — сделать одну из самых качественных и атмосферных фанатских игр по FNAF.

## 2. Основные особенности
- Полностью 3D (в отличие от классического 2D FNAF)
- Реалистичная скелетная анимация и ретаргетинг
- Динамическое освещение и звук
- Нелинейный сюжет + несколько концовок
- Система выживания и скрытности

## 3. Сюжет (полная версия)
Подробный лор, предыстория, ключевые персонажи, временная линия.

## 4. Геймплейные механики (разложено по пунктам)
- Система ночей (1-7 + кастомные)
- Управление охранником (движение, фонарик, двери, мониторы)
- ИИ аниматроников (разные модели поведения)
- Система энергии и расхода батареи
- Скрытность и шум
- Мини-игры и пасхалки

## 5. Аниматроники (детальное описание каждого)
- Freddy
- Bonnie
- Chica
- Foxy
- Golden Freddy
- Новые оригинальные персонажи

## 6. Техническая архитектура
- Почему Python + Ursina
- Asset pipeline (FBX → glTF + ретаргетинг)
- Структура проекта
- Планируемые системы

## 7. Roadmap (этапы разработки)
- Phase 1: Core Engine
- Phase 2: Первая ночь
- Phase 3: Полная игра
- Phase 4: Полировка и релиз
